# List 

List adalah urutan elemen yang teratur <b>(ordered)</b> dan boleh berisi <b>duplikat</b>.

## ArrayList

ArrayList merupakan turunan List yang pendekatannya seperti <b>array namun dinamis</b>.

ciri khas:
- cepat untuk baca dengan index (array.get(0))

kapan dipakai:
- saat data banyak diakses daripada diubah.

## LinkedList
LinkedList merupakan turunan List yang berisi Node berantai.

ciri khas:
- cepat untuk insert/delete di awal atau ditengah
- lambat untuk akses menggunakan get(x)

kapan dipakai:
- saat sering insert/delete
